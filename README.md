# Calculator API
 This is a simple API built using FastAPI that performs basic arithmetic operations
 ## Features
 -Addition
 -Subtraction
 -Multiplication
 -Division
 ## Installation
 pip install fastapi uvicorn
 ## Run the app
 uvicorn main:app --reload
 ## API Endpoints
 Add
 /add?a=5&b=9 → 14
 Sub
 /sub?a=5&b=9 → -4
 Mul
 /mul?a=5&b=9 → 45
 Div
 /div?a=5&b=9 → 0.45
